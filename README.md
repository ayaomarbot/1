# Проект «Айа» — лонгрид «Работа с Нафсом»

Статья-лонгрид «Как через Работу с Нафсом стать успешной в обоих мирах» — в 4 частях,
красивый адаптивный сайт (открывается с компьютера и телефона). Чистая статика: HTML + CSS +
немного JavaScript, без сборки и зависимостей.

## Структура

```
site/
├── index.html      → Часть 1. Введение «Вас обманули»
├── part-2.html     → Часть 2. Почему это работает (личная история)
├── part-3.html     → Часть 3. Любовь к Аллаху (Ступень 10 + история Камилы)
├── part-4.html     → Часть 4. Доверие и Предназначение + Спринт
├── styles.css      → весь дизайн (тёплый песок + золото)
├── script.js       → прогресс-бар чтения и плавное появление блоков
└── images/         → все изображения (оптимизированы для веба)
```

После каждой части — кнопка «Читать следующую часть →». В конце 4-й части — блок со ссылками
на Спринт (просмотр программы + 2 кнопки оплаты).

## Как открыть локально

Просто дважды кликните по `index.html` — откроется в браузере. Либо запустите локальный сервер:

```bash
cd site
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000` в браузере.

## Как опубликовать на GitHub Pages

1. Создайте репозиторий на GitHub и загрузите туда **содержимое папки `site/`**
   (чтобы `index.html` лежал в корне репозитория).
2. В репозитории откройте **Settings → Pages**.
3. В разделе **Build and deployment → Source** выберите **Deploy from a branch**.
4. Branch: `main`, папка `/ (root)`. Нажмите **Save**.
5. Через 1–2 минуты сайт будет доступен по адресу вида
   `https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПОЗИТОРИЯ/`.

## Как менять текст или ссылки

- **Текст** — прямо в `.html`-файлах (открываются любым редактором).
- **Ссылки на оплату/Спринт** — в конце `part-4.html`, блок `<section class="cta">`.
- **Цвета** — в начале `styles.css`, блок `:root` (например, акцент `--gold`).
- **Картинки** — в папке `images/` (имена уже подставлены в HTML).
